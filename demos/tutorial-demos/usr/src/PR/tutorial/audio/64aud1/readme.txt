Operation

* Use the up/down keys on the cross-key to move between menu items
* Use the right/left keys on the cross-key to change values 
* In the SONG and SE menus, the A button will play back the selected song or sound effect 
* Use the B button to stop the voice whose play back was started last
* Use the Z trigger to stop all sounds
* In the FO (FadeOut) menu, lef/right on the C button changes values 10 at a time
